
Uveb
================================

.. toctree::
   :maxdepth: 2

   usage/index

Uveb is the `REST
<https://en.wikipedia.org/wiki/Representational_state_transfer>`_ API for the
UWC CSC VR Application. Uveb is responsible for providing all of the dynamic
data to the frontends that require it.
